# QA 체크리스트

직접 플레이하면서 아래 항목을 순서대로 확인한다.

---

## 1. 초기 상태

- [ ] 전사 / 성직자 / 궁수 3명이 필드에 배치된 상태로 시작된다
- [ ] 각 캐릭터 카드에 `Lv1`, `0/3 재료`, `1/1 배치중` 표시가 맞다
- [ ] 적이 필드 반대편에 스폰되어 있다

---

## 2. 이동 및 공격

- [ ] 캐릭터 클릭 시 이동 가능 범위(파란 셀)가 표시된다
- [ ] 이동 후 공격 범위 내 적에게 자동으로 데미지가 들어간다
- [ ] 같은 턴에 이동한 캐릭터는 다시 선택되지 않는다 (테두리 링 사라짐)
- [ ] 모든 캐릭터가 이동하면 자동으로 적 턴으로 넘어간다
- [ ] 확인 버튼으로도 적 턴을 수동으로 넘길 수 있다

---

## 3. 4명 배치 시 턴 종료 버그 확인 (수정됨 - 2026.06.14)

> 조건: 해금으로 아군 4명이 필드에 배치된 상태

- [ ] 아군 4명 중 3명만 이동해도 턴이 넘어가지 않는다
- [ ] 배치(드래그)한 직후의 캐릭터도 반드시 이동해야 턴이 넘어간다
- [ ] "턴 종료" 버튼으로는 언제든 수동으로 턴을 넘길 수 있다

---

## 4. 동일 종류 캐릭터 이동 버그 확인 (기존 버그)

> 조건: 해금으로 전사를 2명 보유한 상태

- [ ] 전사 1번을 이동시킨 후 전사 2번도 이동 가능하다 (선택 가능, 링 표시)
- [ ] 전사 2번을 먼저 이동시킨 후 전사 1번도 이동 가능하다

---

## 4. 뽑기 및 강화 재료

- [ ] 뽑기 버튼(티켓 1장 소모)으로 랜덤 재료가 추가된다
- [ ] 카드 하단 점 3개가 재료 수에 맞게 채워진다
- [ ] 재료 3개가 쌓이면 해당 캐릭터가 Lv2로 강화된다 (필드에 있는 캐릭터 기준)
- [ ] 강화 후 카드에 `Lv2`, 재료 `0/3`으로 초기화된다
- [ ] 필드에 없을 때 재료 3개가 쌓이면 roster 대기 캐릭터가 강화된다
- [ ] 그것도 없으면 Lv2 캐릭터가 새로 roster에 추가된다

---

## 5. 배치 비용 (Lv1 재료 소모)

- [ ] 대기 중인 캐릭터의 배치 버튼에 `배치 💎N` 형태로 현재 Lv1 재료 수가 표시된다
- [ ] Lv1 재료가 0개일 때 배치 버튼이 비활성화된다
- [ ] Lv1 재료가 1개 이상일 때 배치 버튼이 활성화된다
- [ ] 배치 시 Lv1 재료가 1 감소한다
- [ ] 티켓은 소모되지 않는다
- [ ] Lv2 캐릭터를 배치해도 Lv1 재료 1개만 소모된다

---

## 6. 캐릭터 사망 및 부활

- [ ] 캐릭터가 HP 0이 되면 필드에서 사라지고 roster 대기 상태가 된다
- [ ] 죽은 캐릭터의 카드에 `0/1 배치중` (대기 중)으로 표시가 바뀐다
- [ ] Lv1 재료가 있으면 죽은 캐릭터를 다시 배치할 수 있다 (배치 버튼 활성화)
- [ ] Lv1 재료가 0개면 배치 버튼이 비활성화된다 (부활 불가)
- [ ] 부활 후 레벨이 유지된다 (Lv2가 죽었으면 Lv2로 부활)
- [ ] 부활 후 HP가 만피로 회복된다

---

## 7. 슬롯 해금

> 조건: 티켓 3장 이상 보유

- [ ] 해금 버튼(티켓 3장 소모)으로 랜덤 타입 캐릭터가 roster에 추가된다
- [ ] 추가된 캐릭터의 레벨이 기존 동일 타입 캐릭터의 최고 레벨과 같다
  - 예: 전사 Lv2가 있는 상태에서 해금으로 전사가 추가되면 Lv2로 추가됨
- [ ] 해금된 캐릭터의 HP가 레벨에 맞게 설정된다 (Lv2: 120, Lv3: 140)
- [ ] 총 4명이 되면 해금 버튼이 비활성화된다

---

## 8. 적 턴

- [ ] 적이 이동하고 인접한 아군을 공격한다
- [ ] 데미지 이펙트(빨간 구체)가 표시된다
- [ ] 아군 HP 바가 깎인다
- [ ] 성직자 적은 주변 적을 회복한다 (녹색 이펙트)
- [ ] 적 처치 시 티켓이 +1된다

---

## 9. 웨이브 클리어 및 게임 종료

- [ ] 적을 전부 처치하면 다음 웨이브가 시작된다
- [ ] 웨이브 수가 HUD에서 증가한다
- [ ] 아군이 전부 사망하면 GAME OVER 화면이 표시된다
- [ ] 재시작 버튼이 동작한다

---

## 10. 웨이브 5 보스 클리어 (1스테이지)

> 조건: 웨이브 1~4를 클리어하여 웨이브 5 진입

- [ ] 웨이브 5 시작 시 보스(👹)가 그리드 중앙 상단(3, 8)에 스폰된다
- [ ] 보스가 일반 적과 함께 등장한다
- [ ] 보스의 HP 바가 표시된다 (HP 500)
- [ ] 보스를 처치하면 "1스테이지 클리어" 화면이 표시된다
- [ ] 웨이브 6으로 진행되지 않고 게임이 종료된다
- [ ] 클리어 화면에 처치 수, 웨이브 정보가 표시된다
- [ ] 재시작 버튼이 동작한다

---

## 11. HP / 공격력 표시 (미구현 확인)

- [ ] 현재 캐릭터별 HP 숫자, 공격력 수치가 UI에 표시되지 않는다 (미구현 상태 확인용)
- [ ] 적의 HP도 숫자로 표시되지 않는다 (미구현 상태 확인용)
